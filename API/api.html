VOD Library Web-service API

The web-service allows you to get movies data from JSON File

Instructions

Method: GET
URL Example: /getAllMovies

Get all Movies that are reletad to the specific Holiday

Method: POST
Param: holiday_name
Example Possible Holidays: Passoveer, Christmas, Shavuot, Yom Kippur
URL Example: /getMoviesByHoliday/Christmas
Response: Movies That are Belonged to the Holiday

Get all Movies that are reletad to the specific Holiday and Language

Method: GET
Param: holiday_name, language
Example Possible Holidays: Passoveer, Christmas, Shavuot, Yom Kippur
Example Possible Languges: English, Hebrew
URL Example: /getMoviesByHolidayAndLanguage/Passover/English/
Response: Movies That are Belonged to the Holiday and in asked language
